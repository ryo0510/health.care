# Health Care

## サイト概要
日々の健康状態の記録をできるサイト

### サイトテーマ
健康状態の記録。

### テーマを選んだ理由
コロナにより在宅時間も増えて、健康に気をつかうことがとても増えてきたと思います。<br>
そこで毎日の健康状態を記録でき、健康管理についての豆知識などを共有できるようなサイトを作りたいと思い今回このテーマを選びました。

### ターゲットユーザ
生活習慣が乱れがちな社会人。特に20代。

### 主な利用シーン
積極的に生活習慣の改善を目指したい時。
